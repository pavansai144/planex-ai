# PLANEX AI

AI-powered lunar crater detection system using computer vision.

## Features
- Lunar crater detection
- Edge detection pipeline
- Image preprocessing
- Mission dashboard UI
- Interactive chatbot
- 3D rotating moon visualization

## Technologies
- Python
- Flask
- OpenCV
- HTML / CSS / JS
- Three.js

```bash
pip install -r requirements.txt
python app.py
